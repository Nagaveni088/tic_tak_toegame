# Tic Tac Toe Game

## Description
This project is a Tic Tac Toe game developed as a mini project using Python. It demonstrates basic game logic, control flow, and user interaction.

## Features
- Two-player Tic Tac Toe game
- Simple and easy-to-understand rules
- Console-based implementation
- Clear game flow and logic
